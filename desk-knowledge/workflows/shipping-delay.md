# Werkwijze: verzending / vertraging

## Wanneer

Klant vraagt waar het pakket blijft, tracking, of leverdatum.

## Stappen

1. Koppel order op e-mail / ordernummer (automatisch in tool).
2. Lees Shopify-verzendstatus + **live DPD** (indien beschikbaar).
3. Als nog niet verzonden (`unfulfilled` / geen tracking): leg productie/levertijd uit uit ordercontext — geen vaste dag beloven.
4. Als wel DPD-data: geef actuele status, pakketnummer, verwijs naar dpd.nl.
5. Geen excuus-romans; één duidelijke status + wat klant nu kan doen.

## Niet doen

- Geen andere carrier-status verzinnen.
- Geen "we kunnen je order niet vinden" als order wél gekoppeld is.
