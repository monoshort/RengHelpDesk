omlijst via GitHub Pages Repo

GitHub Pages draait geen Node-server. De helpdesk blijft op je host draaien (bijv. Render). Hier plak je die https-URL één keer; daarna opent de helpdesk in het venster hieronder.

Tip: je kunt ook eenmalig ?url=https%3A%2F%2F… in de adresbalk zetten. Als de live site X-Frame-Options blokkeert, zie je een leeg vlak — open de URL dan rechtstreeks.
